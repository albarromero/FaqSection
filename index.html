Preguntas frecuentes

¿Qué es la programación web?

El desarrollo web es un término que define la creación de sitios web para Internet.

¿Qué es el Frontend?

Hace referencia al desarrollo para la renderización de contenido del lado del cliente. Un proyecto web ejecuta, a nivel de servidor, todas las operaciones lógicas y, en el navegador del cliente, se renderiza lo procesado en el servidor para crear la representación gráfica.

¿Qué es el Backend?

Backend se refiere a todos los lenguajes y herramientas que intervienen en la renderización de cualquier proyecto en el lado del servidor. El desarrollo backend no solo está orientado a la creación de páginas web, también se usa para el procesamiento de bases de datos, plataformas, automatizaciones, servicios en la nube, etc.

¿Qué hace un desarrollador Web?

Un desarrollador web es el encargado de crear y mantener cualquier proyecto (páginas web, plataformas y aplicaciones…). Existen desarrolladores enfocados en frontend y otros en backend, pero el mercado actualmente demanda más el perfil Full Stack, es decir, que dominan ambas partes.
